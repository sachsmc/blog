For my first post, I'd like to write a brief tutorial on how I made the cool interactive “Brownian Viaducts” as seen on my homepage. There were a number of steps invovled in getting from the idea to the d3 implementation that I plan to work through. But first, what is a Brownian Viaduct?

Brownian Bridges and Viaducts

A Brownian bridge is a Gaussian process indexed by (0, 1) that is pinned down to 0 at both ends. In statistics, Brownian bridges come up when we talk about the empirical process, that is the empirical cumulative distribution function (cdf).

To solidify things, lets consider a sample from the Uniform distribution on (0, 1). The empirical cdf at t is simply the proportion of samples that are less than t.

{% highlight r %} mean(runif(100) < 0.25) {% endhighlight %}

{% endhighlight %}

Here I took t = 0.25, but the empirical process is just the random function of t. Let's plot it:

{% highlight r %} empcdf.unif <- function(t) { sapply(t, function(x) mean(runif(100) < x)) } curve(empcdf.unif, from = 0, to = 1) {% endhighlight %}
